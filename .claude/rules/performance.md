Three.js: 3-tier performance system in HeroScene.tsx (high/medium/low) based on devicePixelRatio, hardwareConcurrency, deviceMemory. Low tier: no post-processing, 1200 particles, DPR 1, low-power GPU. Medium: bloom + vignette only, 2500 particles. High: all effects (bloom, chromatic aberration, vignette, noise), 4000 particles, multisampling 4. FontAwesome: global library.add() removed — icons imported directly per component for tree-shaking. Blog grid: lazy loads 6 posts at a time via IntersectionObserver. Mobile gets compact card layout (PostCardCompact). prefers-reduced-motion: all infinite CSS animations disabled. SmoothScroll uses Lenis + GSAP (185KB) — loaded globally, future optimization target.
