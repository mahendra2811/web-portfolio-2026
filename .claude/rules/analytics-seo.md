Analytics: GA4 via @next/third-parties + custom BlogAnalytics (scroll depth, reading time, completion %). GTM optional via NEXT_PUBLIC_GTM_ID. Search Console verification via NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION meta tag. Every page MUST have: metadata export with title + description, canonical URL (alternates), OpenGraph tags. Blog posts have BlogJsonLd (BlogPosting + BreadcrumbList schema), visible breadcrumbs, author byline, related posts, TableOfContents. Sitemap auto-generated by next-sitemap (runs in build script). RSS feed at /feed.xml. For client-component pages (projects, skills, contact), metadata goes in layout.tsx not page.tsx. Blog images use Sanity CDN or Unsplash — avoid uploading to Sanity to preserve free plan asset quota. prefers-reduced-motion CSS disables all infinite animations.
