Blog automation is set up via scripts/blog.ts. When user says "create a blog about X" — generate a .md file in content/blog/ with frontmatter (title, excerpt, categories, coverImage), then run `npx tsx scripts/blog.ts publish content/blog/<filename>.md`. For bulk: `npm run blog:batch`. Blog writing style: senior dev voice, emojis throughout, tables for comparisons, humor, opinionated, first-person perspective. Never write dry/textbook content. Always use frontmatter with title, excerpt, categories. Cover images use Unsplash URLs (https://images.unsplash.com/photo-xxx?w=1200&h=630&fit=crop&auto=format). The script handles markdown-to-portable-text conversion, category auto-creation, and duplicate detection by slug.
